# QnA
Question and Answer API

# Routes
#### GET all product Q&As:

localhost:3003/qa/

#### GET one product Q&A:

localhost:3003/qa/:product_id

#### POST one product Q&A:

localhost:3003/qa/:product_id

#### DELETE one product Q&A:

localhost:3003/qa/:product_id

# Generate 10M Records

`bash serverSDC/seed.sh 10000000`
